================================================================================
                         FEATURE SUMMARY
================================================================================

REGIME FEATURES (4):
  - regime_encoded: 0-3 (CHOPPY, NORMAL, TRENDING, VOLATILE)
  - vol_ratio: Volatility ratio (0-2)
  - One-hot encoded to 4 binary features for RandomForest

TECHNICAL FEATURES (3):
  - rsi_strength: Current RSI (0-100)
  - rsi_div_regular: Regular divergence flag (0/1)
  - rsi_div_hidden: Hidden divergence flag (0/1)

PATTERN FEATURES (1):
  - macd_div: MACD divergence flag (0/1)

MARKET CONTEXT FEATURES (2):
  - sentiment_fg: Fear/greed index (0-100)
  - mtf_bias: Multi-timeframe bias (-1/0/1 for SHORT/NEUTRAL/LONG)

SETUP QUALITY FEATURES (3):
  - confluence_score: Normalized confluence (0-1)
  - entry_to_sl: SL distance in ATRs (0-5)
  - direction_encoded: Direction (-1/0/1)

TEMPORAL FEATURES (1):
  - duration_candles: Hold duration estimate (int)

TOTAL: 12 base features → 14 with regime one-hot encoding

================================================================================
                         ERROR HANDLING
================================================================================

SCENARIO: Model file missing
  Output: ⚠️  ML model load failed: [Errno 2] No such file
  Fallback: load_model() returns False
  Behavior: score_with_ml() returns 0.5 (neutral confidence)
  Result: Trading continues normally

SCENARIO: Invalid feature values (NaN/inf)
  Handling: Replaced with 0 in feature list
  Result: Scoring continues without error

SCENARIO: ML scoring exception
  Output: ⚠️  ML scoring error: {exception message}
  Fallback: Returns 0.5 confidence
  Result: No crash, graceful degradation

SCENARIO: Missing setup keys
  Handling: Defaults used (0 for binary, 50 for sentiment, 1.0 for vol_ratio)
  Result: Scoring continues with conservative defaults
